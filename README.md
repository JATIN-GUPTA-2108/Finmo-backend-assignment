## INITITALIZE

```
$  run npm install
```
## SET UP ENV FILE
PORT
DATABASE_URL
JWT_SECRET
ALPHAVANTAGE_API_KEY

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
